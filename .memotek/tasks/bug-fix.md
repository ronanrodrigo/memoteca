## Fluxo de Fix
1. Intake — Coletar informações do bug
2. Diagnóstico — Identificar causa raiz
3. Fix — Implementar correção
4. Testes — Validar que o bug foi corrigido
5. PR — Criar e mergear correção
6. Memory — Atualizar issue
